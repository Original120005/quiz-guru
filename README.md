# Quiz Guru

**Интерактивная платформа с квизами, рейтингами и генерацией вопросов через ИИ (Grok)**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

</div>

---

## Особенности

- Регистрация и вход (JWT)
- Личный профиль: статистика, аватар, бейджи
- Квизы по темам: наука, IT, мемы, кино
- Поиск и фильтры
- Баллы за ответы + бонусы за скорость
- Рейтинг игроков (все время / месяц)
- Генерация квизов через ИИ (Grok)
- Адаптивный дизайн (SCSS)
